A) Requirements

  • PHP 5.6 or higher

B) Installation

  1. Import a Database File

    2. Find the tanzame.sql file in project main folder.



    Create database in PHPMyAdmin and import database file into your database.



  2. Open a config/database.php file.



  3. Change database user, password and name in database.php File

  4. Uploading to Server

    • Make a zip file of updated project main folder using any compressor software like Winrar
    • Log onto your Web space either via FTP client like cuteFTP / FileZilla or CPanel using your webserver username and password.
    • Browse to the directory where you wish to place your script.
    • It is advisable to choose a folder where you will want to install the script.
    • it is preferable to choose your webhost root folder popularly name “public_html” or “htdoc”.
    • Select zip file using the upload button in your FTP client or Cpanel, then browse the drive/folder where the zip file is.
    • Click “Upload,” “Send,” or your FTP client’s equivalent.
    • Wait until the transfer is complete.
    • Unzip the zip file after uploading.
